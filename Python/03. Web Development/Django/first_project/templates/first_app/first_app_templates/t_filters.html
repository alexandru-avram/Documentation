Template Filters

Template filters are items attached to the view corresponding to a page. They are usually cread in the app/views.py file, and pass into a dictionary. Afterwards, they can be along with either a built-in Django code, or one created in the app/templatetags folder.

Examples below:

{{ text|upper }}
{{ text|cut:"context"}}
