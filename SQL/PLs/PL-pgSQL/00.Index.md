# PL/pgSQL — Course-Style Notes

This set mirrors your Postgres course modules. Each note is concise and example-first.

1. **Server programming** → environment, security, search_path, plan caching, logging.
2. **Control flow (Functional programming)** → `IF`, `CASE`, loops, `FOREACH`, `EXIT/CONTINUE`.
3. **Functions** → scalar/SET-returning, parameters, volatility, security, performance.
4. **Stored Procedures** → `CALL`, transaction control inside procedures.
5. **Cursors** → implicit vs explicit, `refcursor`, `FETCH`, performance tips.
6. **Triggers** → row vs statement, BEFORE/AFTER/INSTEAD OF, transition tables.
